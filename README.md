# Molotov 

It's a fast-paced 2D top-down survival shooter made with **Phaser 3**. Defend against enemy waves while scavenging for gasoline to fuel your escape vehicle.

##  About
In a hostile urban environment, your mission is to survive and escape. Scavenge for supplies, fight off armed enemies, and siphon gasoline from abandoned cars to fill your orange getaway vehicle.

##  Features
- **Combat**: Face 4 types of enemies with different behaviors and weapons (Melee, Pistol, Shotgun, Rifle).
- **Weapons Arsenal**: Loot and use Pistols, Shotguns, and Assault Rifles.
- **Resource Management**: Manage health, ammo, and fuel capacity.
- **Progression**: Level up difficulty as time passes.
- **Save/Load System**: Continue your progress anytime using local storage.

##  Controls
| Key | Action |
| --- | --- |
| **W, A, S, D** | Move |
| **Mouse Right Click** | Shoot |
| **R** | Reload |
| **SPACE** | Dash / Dodge |
| **E** | Interact (Loot items, siphon gas) |
| **P** | Pause Game |

##  Tech Stack
- **Engine**: Phaser 3
- **Language**: JavaScript
